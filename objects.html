JavaScript Objects

The following code will Create an object and display its properties manually using the dot operator 

var person = {
    firstName : "Amy",
    lastName : "Williams",
    age : 20,
    GPA : 3.95
};

document.getElementById("demo").innerHTML =
person.firstName + " is " + person.age + " years old.";

The following example adds a function called displayData() that tells the object how to do this on its own

var person = {
    firstName: "Amy",
    lastName: "Williams",
    age: 22,
    GPA: 3.95,
    display: function(){
      return this.firstName + " " + this.lastName + "GPA: " + this.GPA;
    }
};

var text = person.display();
