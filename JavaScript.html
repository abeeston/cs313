Parameters

There are a few important items to note about paramenters
  • Parameters are the names listed in the function definition
  • Arguements are the real values passed to and received by the function
  • Arguments are passed by value
    • If a function changes an argument's value, it does not change the paramenter's original value
  • Objects are passed by reference
    • If a function changes an object's value, it will change the paramenter's original value
  • Type checking is not performed on passed arguments
  • JavaScript functions do not check the number of arguments received

In functions where there are missing arguments (less than declared) the missing values are set to undefined 

function display(x, y)
{
	document.getElementById("results5").innerHTML = x + ", " + y;
}
display(2);

Results:

JavaScript functions have a built-in object called the argument object that can be used to find the arguments. 

function displayArgs() {
   	var text = "";
   	var i;
   	for (i = 0; i < arguments.length; i++)
   	{
       	text += "  " + arguments[i];
   	}
   	return text;
}
	
var x = displayArgs(1, 10, 500, 115, 72, 64);

Arrays

Important points
  • Arrays are JavaScript object
  • Arrays use numbers to access their elements
  • Arrays may contain multiple data types
  • Arrays are zero-relative
  • Arrays have a length property that can be used while looping through them

The following example shows mixing student names with a student id

var text = "";
var students = [
	"Amy Williams",
	552359831,
	"Kevin Beesley"
];

for (var i = 0; i < students.length; i++)
{
	text += " " + students[i];
}

Results:

The following example shows how JavaScript can create an array using "new"

var text = "";
var students = new Array("Trevor Walker", 989843536, "Joseph Brown");

for (var i = 0; i < students.length; i++)
{
	text += " " + students[i];
}

Associative Arrays

Associative arrays may be used as follows. However, it is important to note that these arrays cannot be accessed with the normal index as follows:

var student = [];
student["firstName"] = "Amy";
student["lastName"] = "Williams";
student["age"] = 22;

var text = "";
text += student[0];

Results:

Associative arrays may be accessed through the dot operator as follows: 

var student = [];
student["firstName"] = "Amy";
student["lastName"] = "Williams";
student["age"] = 22;

var text = "";
text += student.firstName + " " + student.lastName + " " + student.age;

Results:

The following demonstrates another way that JavaScript arrays can be defined to use key/value pairs as well

var text = "";
var student = {firstName:"Amy", lastName:"Williams", age:22};
text += student.firstName + " " + student.lastName + " " + student.age;
